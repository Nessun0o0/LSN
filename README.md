# NUMERICAL SIMULATION LABORATORY 2025/2026

**Author: Francesco Palombo**

This repository contains the exercises for the numerical simulation laboratory course.

All the notebooks containing the figures and the comments to the exercises are in the folder [Notebooks](Notebooks/). Each lesson has its own jupyter notebook and its own folder where the C++ code and the output data is located. 

Every exercise can be compiled by entering in the corresponding folder and executing `make`.
